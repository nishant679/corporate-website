과거로 여행을 떠납니다.
우리가 해낸 프로젝트들의 쇼케이스.

우리는 미래에 대해서 말하기를 좋아하지만, 여기서는 과거를 살핍니다. 프로젝트를 세세하게 분석한 작업 지시서부터 인터페이스 디자인, 그리고 모사코드 알고리즘까지, 우리가 가장 자랑스럽게 여기는 프로젝트들을 소개합니다. 더 많은 프로젝트들을 보기 위해서는, 회사 미디움 블로그나 유투브 브이로그를 방문하세요.

호랑이의 영혼을 담은 인공지능

요구사항: 프로젝트 고타이거에서는, 이 세상에 존재하지 않는 럭셔리 매트리스 큐레이팅 서비스를 구상하고, 처음부터 설계, 디자인, 개발하여 웹, iOS, 그리고 Android에 호환되는 애플리케이션을 배포해야 했습니다. 주요 장애 요소로는, 매트리스 생산설비가 미완성인 상황에서 최종 서비스의 모습을 예측하며 개발을 해야 한다는 점이었습니다.

테크: Reactjs, React Native, Node.js, Redis, Wordpress Rest APIs, etc
기간: 5 개월
팀 리드: Hemprasad Badgujar (테크), Vishal Katral (디자인)
팀 구성: Ashish Tiwari, Nitin Kumar, Akash Mittal, Ashu Pal, Raj Yadav, Sandeep Yadav, Harswardhan Kumar

소셜 미디어 사다리를 크롤링하는 거미.

요구사항: 프로젝트 래더에서는 소셜 미디어 인플루엔서들에 대해 공개된 정보를 취합하고, 데이터를 분석한 후, 이를 광고주들에게 제공하는 유망한 한국 스타트업과 함께 일했습니다. 우리가 담당한 주요 업무로는 Node와 Python으로 이러한 데이터가 신속하게 업데이트가 될 수 있는 알고리즘을 개발하는 것이었습니다.

테크: Node, Python (Beautiful Soup, Scrapy)
기간: 3 개월
팀 리드: Hemprasad Badgujar
팀 구성: Sumit Patel, Pratul Singh

Efficient internal notifications

Requirements: For this project we built an internal push notification system using AWS serverless lambdas that HumaxIT uses heavily to notify employees for real time events. Our challenges included building a system with minimum latency and maximum efficiency.

Tech: AWS Services, React Native, FCM, APNS
Duration: 2 Weeks
Leads: Anand Rai
Team: Raj Yadav, Ashish Tiwari, Nitin Kumar
